#+title: Dev Flakes

This is a repository for direnv template flakes so I can quickly setup a project
with a certain language.

*  How to Use
for example, to install a python3.13 flake simply run:
#+begin_src bash
wget https://raw.githubusercontent.com/Zivision/dev-flakes/main/base-languages/python/313/flake.nix
#+end_src
In the root of a project. Then define your .envrc and enable direnv
#+begin_src bash
echo "use flake" >> .envrc
direnv allow
#+end_src
